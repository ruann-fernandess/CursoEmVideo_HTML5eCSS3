Principais Formatações:

Negrito/ Destaque

Nesta frase temos um termo em negrito utilizando a tag <b> </b> (não semântica)

Nesta frase temos um termo em destaque utilzando a tag <STRONG> </STRONG> (semântica)

Itálico/ Enfâse

Nesta frase temos um termo em itálico utilizando a tag <i> </i> (não semântica)

Nesta frase temos um termo em enfâse utilzando a tag <EM> </EM> (semântica)

Texto marcado

Podemos criar também um texto marcado usando a tag <MARK> </MARK>

Podemos criar outro texto marcado usando a tag <MARK> com <STYLE>

Texto grande e pequeno

Estamos criando um texto grande e um texto pequeno neste parágrafo com as tags <small> </small> e <big> </big>

Texto deletado

Podemos marcar um texto como excluido para marcar que ele deve ser lido mas não considerado, com a tag <del> </del>

Texto inserido

Podemos macar um texto como inserido para da enfâse ao indicar que o mesmo foi inserido depois, com a tag <ins> </ins>

Texto Sobrescrito

Para inserir coisas do tipo x2 + 3 = 6, com a tag <sup> </sup>

Texto Subscrito

Para inserir coisas do tipo H20, utilizando a tag <sub> </sub>
